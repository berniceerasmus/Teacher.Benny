Multiple Choice

📘 Grammar Focus — Choosing the Right Tense

Present Simple

Subject + V / V-s

Routines and facts. Use with every day, always, usually.
→ She checks the line every morning.

Present Continuous

am / is / are + V-ing

Actions happening at this moment. Use with right now, currently, at the moment.
→ I am having a meeting right now.

Past Simple

Subject + V-ed / irregular

Finished actions. Use with yesterday, last week, at 3 a.m., in 2020.
→ The system crashed at 3 a.m.

Present Perfect

have / has + past participle

Recent results or news — no specific time. Use with just, already, yet.
→ We have finalised the contract.

Passive (Present Simple)

is / are + past participle

The action is more important than who does it.
→ The software is updated every Monday.

Zero Conditional

If + present simple, present simple

Fixed rules — automatic results.
→ If the temperature rises, the machine stops.

Error Correction

📘 Grammar Focus — Common Error Types

Each sentence has exactly one error. The mistakes come from these areas — look for them carefully:
Wrong tense with a time word (e.g. yesterday + present perfect) Wrong auxiliary (e.g. don't instead of doesn't) Missing be in passive voice will in the if-clause of a conditional Wrong tense with since (needs present perfect, not present continuous) Past simple with yet (needs present perfect)

💡 Key rule: Present perfect (have/has + past participle) is never used with finished time words like yesterday or last week. Use past simple instead. But with since and yet, you need present perfect.

Word Bank

📘 Grammar Focus — Modal Verbs

must / have to

Strong obligation

must = rule, law, or strong personal obligation.
have to = external requirement (often a rule from outside).
→ All staff must wear PPE.

should

Advice / recommendation

It is a good idea, but not compulsory.
→ You should ask IT for help.

could

Ability or gentle suggestion

A possibility or option — softer than should.
→ You could try a different approach.

might

Possibility (not certain)

Something is possible but you are not sure.
→ The delivery might be late.

can

Ability / general possibility

You have the skill or it is physically possible.
→ I can speak English in meetings.

⚠ All modals are followed by the base form of the verb (no to): must wear, should send, might arrive.

Matching

📘 Grammar Focus — Functional Language in the Workplace

In professional English, we use fixed phrases to communicate specific functions. The grammar inside the phrase gives you a clue about its purpose:
"I see your point, but…" → acknowledge + disagree politely "Could you repeat…?" → modal question = polite request "…happened because…" → cause and explanation "We have already…" → present perfect = recent completed result "I suggest that we…" → suggestion (verb + that + clause) "In my opinion…" → personal view / opinion "First… then… finally…" → sequencing steps in a process

Sentence Building

📘 Grammar Focus — English Word Order

Standard Order

Subject + Verb + Object

→ The team is testing the new software.

Passive Order

Subject + is/are + past participle + by + agent

→ All parts are inspected by the quality team.

Negative Perfect

Subject + have/has + not + past participle + (yet)

→ We have not finished the installation yet.

Conditional

If + [condition], + [result]

→ If we miss the deadline, the order closes today.

Transform the Sentence

⏱ ~6 min
Transformation

📘 Grammar Focus — How to Transform Sentences

Active → Passive (Present)

Object + is/are + past participle + (by + agent)

Active: The engineer checks the equipment.
Passive: The equipment is checked (by the engineer).

Active → Passive (Past)

Object + was/were + past participle

Active: We delivered the order.
Passive: The order was delivered.

Past Simple → Present Perfect

Subject + have/has + past participle

Past: The team finished the report.
Perfect: The team has finished the report.

Possibility → Modal might

Subject + might + base verb

It is possible that they will be late.
→ They might be late.

Present Simple → Present Continuous

Subject + am/is/are + V-ing

Routine: We check the data every morning.
Right now: We are checking the data.

must → have to

Subject + have/has to + base verb

All visitors must sign in.
→ All visitors have to sign in.

Free Writing

📘 Grammar Focus — Combining Structures

Good workplace writing uses different tenses together to tell a complete story. Try to include at least three of these:
Past simple → what happened Present perfect → what has been done Present continuous → what is happening now Modal verb → what should / must happen next Passive voice → describing a process formally Conditional → if… then…

💡 Example: "The machine stopped because a part was damaged. We have contacted the supplier and the parts might arrive tomorrow. If they don't arrive, we will need to stop production."
