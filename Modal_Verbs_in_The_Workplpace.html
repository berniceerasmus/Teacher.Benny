Workplace English — Lesson Reference Card

📘
Core Grammar Structures 6 tenses + modals ▾

Present simple

Daily routines, job descriptions

  • I check / I prepare / I install
  • We work with…
  • The system runs…
  • The team handles…

Present continuous

Current work, live updates, meetings

  • We are fixing the issue
  • I am working on…
  • They are testing the system

Past simple

Incidents, reports, audits

  • The machine stopped
  • We checked the system
  • We solved the problem

Present perfect

Progress updates in meetings

  • We have finished the report
  • We have identified the problem
  • We have not received the parts yet

Conditionals

Problem-solving language

  • If the machine breaks, we stop production
  • If we had more staff, we would finish faster

★ Passive voice

Process language — engineering & QC

  • The part is inspected
  • The report was completed
  • The system is updated

★ Modals — highest priority

Ability

cancannot

Advice

shouldshouldn'tcould

Obligation

mustmust nothave to

Probability

mightmaycould

We should contact the supplier  ·  We must check the system again  ·  This might cause a delay

💬 Student questions

What do you do every day at work? Describe your routine using three present simple sentences.
Something went wrong at work last week. Can you tell me what happened? Use past simple.
You notice a machine is making a strange noise right now. What do you say to your manager?
You need your colleague to do something urgently. Which modal would you use — must, should, or could? Why?
The new system was installed yesterday. How do you describe this using passive voice?
💬
Functional Workplace Phrases meetings · problems · solutions ▾

Meetings

  • Let's start
  • I agree / I disagree
  • In my opinion…
  • The main issue is…
  • We need to focus on…

Problems

  • There is an issue with…
  • Something is wrong with…
  • The system is not working
  • We need to fix this quickly

Solutions

  • We should try…
  • A possible solution is…
  • We could also…
  • Let's check…

Clarifying

  • What do you mean?
  • Can you explain that again?
  • Could you repeat that?
  • Do you mean…?

Reporting

  • The problem started when…
  • First we noticed…
  • After that…
  • Finally…

High-Value Real English Patterns use every lesson ▾
Explain a processFirst we…, then we…, after that…, finally…
Report a problemThe issue happened because…
Suggest a solutionI suggest that we…
Give an opinionI think this is important because…
Agree professionallyThat makes sense.
